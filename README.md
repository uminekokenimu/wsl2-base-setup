# WSL2 Base Setup for Development

WSL2ベースイメージ作成用のセットアップスクリプト集

## 概要

このリポジトリは、再利用可能なWSL2ベースイメージを作成し、複数のプロジェクト環境を効率的に管理するためのツールです。

### 特徴

- ✅ Docker（公式インストール方法）
- ✅ Git、keychain
- ✅ 追記形式の設定（既存設定を保護）
- ✅ 簡単にメンテナンス可能

### 構成
```
WSL2ベースイメージ（再利用）
  ↓ export/import
プロジェクト用WSL2インスタンス
  ↓ Dev Container起動
開発環境（プロジェクト固有）
```

## ベースイメージに含まれるもの

- Ubuntu 24.04
- Docker（公式インストール方法）
- Git、keychain
- 基本的な設定（追記形式）

## 含まれないもの（各自で設定）

- SSH鍵
- Git user設定
- デフォルトユーザー設定
- プロジェクト固有の設定
